Conway's Game of Life

A classic cellular automaton — read more on Wikipedia.

Why components?

Every cell in the grid is an individual FuseWire component — either Alive or Dead. When a cell flips state the framework destroys the old component and creates a new one. The grid could run much faster with plain HTML elements, but the point of this demo is to show how fast the FuseWire engine can create, destroy, and render components.
